Run

Run locally

  1. Clone: git clone https://github.com/ianlimescaros/CRM-App.git
  2. Copy env: cp .env.example .env and set DB/LLM keys.
  3. Install PHP deps: composer install
  4. Build CSS: npm install && npm run build:css
  5. Import DB: mysql -u user -p dbname < sql/schema.sql
  6. Serve: php -S localhost:8000 -t public

Want to see it live?

Host on a PHP server (e.g., Hostinger) pointing to public/ and keep .env out of web root.
